
# Slicing Example from Bootstrap Studio to PHP

**The Structure**

```
	app
	|-- assets
	|-- modules 
	|----- home.php
	|----- users.php
	|-- views
	|---- home
	|------ index.php
	|---- users
	|------ index.php
	|---- layouts
	|------ partials
	|-------- header.php
	|-------- footer.php
	|-- index.php
```
