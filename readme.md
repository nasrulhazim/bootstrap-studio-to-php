
# Slicing Example from Bootstrap Studio to PHP

**The Structure**

```
	app
	|-- assets
	|---- bootstrap
	|---- css
	|---- fonts
	|---- img
	|---- js
	|-- modules 
	|----- home
	|-------- index.php
	|-- views
	|---- home
	|------ index.php
	|---- users
	|------ index.php
	|---- layouts
	|------ partials
	|-------- header.php
	|-------- footer.php
	|-- index.php
```
